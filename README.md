# MKisans Repository

This is a multi-dashboard repository for the MKisans project. Each dashboard/module is organized into its own folder to allow multiple team members to work independently.

## Repository Structure

- `kissan_dashboard/`: Full-stack application for the Kisan Dashboard (Backend, Frontend, and App).
- `customer_dashboard/`: (Reserved) Future folder for the Customer Dashboard team.
- `zonal_officer_dashboard/`: (Reserved) Future folder for the Zonal Officer team.

## Getting Started

To work on a specific dashboard, navigate to its folder:

```bash
cd kissan_dashboard
```

### Running the Kisan Dashboard

1. Navigate to the folder: `cd kissan_dashboard`
2. Install dependencies (if not already done): `npm install`
3. Start the development environment: `npm run dev`

## Contribution Guidelines

- Keep your code within your designated dashboard folder.
- Common resources should be discussed before being placed in a shared root folder.
- Ensure you pull the latest changes from the `main` branch before pushing.
